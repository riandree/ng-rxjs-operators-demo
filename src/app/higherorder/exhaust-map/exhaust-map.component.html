This example facilitates an observable of observables. I.e. each event of the "outer" observable is an observable itself that, in this case, represents a limited stream of numbers.

The pipe of the outer observable uses the exhaustMap operator to map the numbers by turning them into a string and concatenating the value of the textfield below.

Each time this button below is pressed a new "inner" observable providing the sequence 0..9 at an interval of 1 second is put into the stream of the "outer" observable.

 

The exhaustMap-Operator subscribes to incoming "inner" observable as long as there exists no active subscription of an earlier "inner" observable. If there is still an active subscription to an "inner" observable any incoming new "inner" observables are ignored and discarded.
